# **App Name**: VibeStream Lite

## Core Features:

- Content Feed: Home screen displaying content cards with thumbnails, titles, and captions for effortless browsing.
- Media Player: Video/audio player screen with floating buttons for 'Save', 'Share', and 'Mini-player' toggles.
- Smart Search: Smart Search tool modal, activated from the player, allowing users to type, speak, or upload a screenshot to initiate a content search.
- Content Discovery: Discover page that allows users to filter content using tags and personalized clips.
- Settings: Settings page allowing users to toggle light/dark modes, manage search history and see app version information.
- Bottom Navigation: Bottom navigation bar for easy access to main app sections (Home, Search, Profile).

## Style Guidelines:

- Light mode background color: White (#F5F6FA) for a clean and calming aesthetic.
- Light mode accent color: Sky-blue (#A7BFFF) to give subtle highlights.
- Dark mode background color: Charcoal (#2C2C2E) for a sleek look.
- Dark mode accent colors: Mint (#68D8D6) and pink (#FFB6C1) for vibrant contrast.
- Font: 'Inter' (sans-serif) for excellent readability on all screen sizes.
- Corner radius: 20px on all major interactive components for a soft, modern feel.
- Consistent use of a 16px grid system for margins and padding to ensure visual harmony.
- Subtle inner and outer shadows to achieve the soft neumorphic effect on buttons and containers.
